Jasmine Darman

Jasmine Darman is a game designer specializing in virtual reality immersion, game design education, and fantasy-driven storylines. She has worked as a project manager on the educational VR platform Reality Flow and has taught game design using VR technology after school at Evan's high school. She has assisted with management and workshops for UCF's downtown maker space, allowing students to gain knowledge on motion capture, 3D printing, and laser cutting for prototyping. She currently works as an instructor of record at UCF for their undergraduate program, teaching students how to use the Unity game engine.

P5Js

Built interactive systems using JavaScript's p5.js framework, including dialogue trees, collision detection, and grid movement mechanics. Published cross-platform games on Itch.io for multiple devices.

2D Unity Engine

Created immersive 2D games using Unity's URP 2D lighting and sprite editor packages. Developed multiplayer games, nonlinear narratives, and rapid mechanic prototypes.

Python Prototypes
